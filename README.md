# Galaxy

> Get info about github repository

```sh
Usage:
  galaxy [command]

Available Commands:
  completion  Generate the autocompletion script for the specified shell
  help        Help about any command
  readme      Prints readme for a repository entered

Flags:
  -h, --help     help for galaxy
  -t, --toggle   Help message for toggle

Use "galaxy [command] --help" for more information about a command.
```
